# Tarea-1-entrega-final
## Funcionamiento:
La tarea consistia en la planificacion de recursos y trabajos teniendo en cuenta eficiencia y restricciones. Para la realizacion de esta tarea se dividio en tres archivos con funciones diferentes que permiten dividir la tarea en problemas mas pequeños. Esta el io-handler, Scheduler y el main. 
- carga de archivos: el usuario debe ingresar los archivos de trabajos y recursos que luego seran utilizados para el cronograma. teniendo en cuenta de que ambos estan en formato JSON y por lo tanto el programa los debe traducir 
- Objetivos: el usuario tiene tres opciones de objetivos de los cuales solo puede escoger uno.
  - maximizar tareas a tiempo
  - minimizar tiempo total
  - minimizar atraso total
 - Output: se prentara un cronograma con las restricciones y requerimientos necesarios para cumplir uno de los objetivos escogidos
 - Datos: finalmente todo debe ser guardado en un archivo JSON
